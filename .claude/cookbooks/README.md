# Cookbooks

Les cookbooks sont des guides techniques structures qui capitalisent les patterns et solutions du projet.

## Utilisation

- **Creation** : `/agent-factory:create-cookbook`
- **Auto-suggestion** : Les cookbooks pertinents sont automatiquement proposes en Phase 0 des taches
- **Format** : Markdown structure avec exemples de code et bonnes pratiques

## Structure d'un cookbook

```markdown
# Titre du Cookbook

## Contexte
## Patterns
## Exemples
## Anti-patterns
## References
```
